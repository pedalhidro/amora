Cadastrar / editar passeio

Gera um phd:tour_<id> conforme ph:TourShape e envia pra POST /upload-tour. Campos com "+ adicionar" suportam criar entidades novas (séries, pessoas, organizadores, provedores) inline.

Identificação
Vira phd:tour_<id>. Edição: digite o ID existente pra sobrescrever.
Upload vai pra tour_assets/<id>/announcement.<ext>. Vira schema:image <URL> no TTL automaticamente.
Séries (uma ou mais) + edição
Cada linha vira ph:inSeriesEdition phd:assoc_<série>_<n>. Vira schema:organizer. Novas: viram phd:org_<slug> a schema:Organization.
Rota
Pessoas Vira prov:wasAttributedTo. Novas pessoas: digite o nome e Enter.
Vira schema:attendee. Independente do contador Participantes abaixo. Vira ph:hasNewcomer. SHACL impede uma pessoa de ser iniciante mais de uma vez na mesma série.
Métricas estimadas
Métricas reais (após o passeio)
Vira ph:departedAt.
Vira ph:arrivedAt.
Vira ph:movingDuration "PT…"^^xsd:duration.
Vira ph:measuredEnergy.
Cada um vira ph:hadBonde true / ph:hadRain true. Marcação ausente = omitido (não negado).
Incidentes Cada linha vira ph:hadIncident "..."@pt (multiplica triples).
